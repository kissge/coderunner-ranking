CODE RUNNERのランキングを見やすくする奴
==================

1. このChrome拡張を導入する
2. ハッピー

Install
-------

1. Google ChromeのSettings -> Extensionsを開く (chrome://extensions/)
2. Developer modeにチェックを入れる
3. Load unpacked extension...をクリックし，manifest.jsonのあるディレクトリを指定する
4. Optionsをクリックし，自分の認証情報を設定する
